 Создаём вектор сразу с тремя готовыми локациями: std::vector<Location> locations(3).
Это как заранее расставить на столе три пустые коробочки с номерами 0, 1 и 2.
Если этого не сделать и сразу пытаться обращаться к locations, программа «споткнётся»,
потому что коробочек ещё нет, а мы уже хотим в них что-то положить.
std::vector<Location> locations(3);
